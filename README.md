# Cài Đặt Mod PVZ2 GE

### Tải và cài đặt Pvz2 GE
- Tải tại link này ```https://pvzge.com/en/download/```

### Cài Đặt Mods
1. Tải/Clone thư mục này về
2. Mở game và bấm ```F10```
3. Qua Tab ```Experimental``` và bật 2 options ```worldmap-json``` và ```plant-level-system```
4. Về lại tab ```Patcher``` và chọn ```Open File Patch Directory```
5. Để 2 thư mục đã tải vào đường dẫn ```com.pvzge.game\gp-next\packs```
6. Vào lại game và chọn ```Save & Reload``` tại mục ```Datapacks```
7. Tắt và mở lại game

### Manual Edit
1. Open file ```plant-levels.json``` in ```Buffz & Neftz Balance Mod\jsons\extensions```
2. Replace ```TIER *``` To ```*```
3. Open file ```plantProps``` in ```Buffz & Neftz Balance Mod\jsons\objects```
4. Find and change Gatling Pea
   a. ```MegaGatling```: ```Peatype & PeaTypePlantFood``` change to ```Pea```
   b. ```Gatling_lvl3```: ```Peatype & PeaTypePlantFood & Cooldown``` change to ```Pea & Pea & 15``` respectively
6. Close and enjoy
